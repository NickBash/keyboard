# Экранная клавиатура

## Пример

Пример доступен по адресу [https://keyboard-3a550.web.app/](https://keyboard-3a550.web.app/)

## Запуск

Вы можете запустить проект

### `npm run start`

И открыть [http://localhost:8080](http://localhost:8080) в браузере

Или собрать

### `npm run build`

Файлы будут в директории /dist
